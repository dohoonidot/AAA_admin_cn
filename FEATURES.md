# AAA 관리자 홈 – 주요 기능 설명

AAA 관리자 웹은 **사용자 관리**, **휴가 총괄**, **선물 발송**, **대화·활동 현황** 등을 한곳에서 처리하는 관리자용 웹입니다.  
로그인 후 좌측 메뉴에서 화면을 이동하며 사용합니다.

---

## 1. 홈

로그인 후 처음 보이는 대시보드 형태의 요약 화면입니다.

| 기능 | 설명 |
|------|------|
| **총 사용자** | 시스템에 등록된 전체 사용자 수 |
| **오늘의 대화** | 오늘 발생한 대화(채팅) 건수 |
| **활성 사용자** | 최근 활동이 있는 사용자 수 |
| **권한** | 현재 로그인한 관리자의 권한(역할) 표시 |
| **AAA 미사용자 명단** | 한 번도 대화 기록이 없는 사용자 목록 (클릭 시 모달로 상세) |
| **사용자 앱 버전 내역** | 직원별 앱 버전(1.3.2, 1.3.1 등) 현황, 버전·부서·개인정보동의 필터, Excel·Markdown 내보내기 |
| **생일자 개인정보 동의 현황** | 생일자 대상 개인정보 동의 여부 조회 및 관리 |
| **추석 선물 모니터링 현황** | 추석 선물 발송·수령 현황 조회, 부서·상태 필터, Excel 내보내기 |
| **최근 활동** | 관리자 로그인·사용자 추가/삭제 등 최근 활동 로그 (접기/펼치기) |
| **권한 안내** | 권한 등급별로 볼 수 있는 메뉴·기능 안내 (권한에 따라 표시) |

---

## 2. 대시보드

기간을 지정해 **대화량·사용자·질문 수** 등을 통계로 보는 화면입니다.

| 기능 | 설명 |
|------|------|
| **기간 필터** | 시작일·종료일 지정 후 «적용»으로 해당 기간 통계 조회 |
| **전체 대화량** | 선택 기간 내 사용자 질문 + AI 응답 수 |
| **활성 사용자 수** | 실제로 질문한 사용자 수 |
| **기타 지표** | 설정된 카드 기준 메시지 수, 사용자 수 등 요약 |
| **차트** | 기간별 대화량·사용자 추이 등 시각화 (Chart.js) |
| **부서별 필터** | 부서 선택 시 해당 부서만 집계 (지원 시) |

---

## 3. 사용자 관리

### 3-1. 재직자

현재 재직 중인 직원을 조회·추가·수정·삭제하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **재직자 목록** | 이름, ID, 부서, 직급, 직책, 관리권한, AAA 조회권한, CSR 권한, **휴가 이력** |
| **검색** | 이름 또는 ID로 검색 |
| **필터** | 부서, 직급, 직책, 관리권한, AAA 권한, CSR 등으로 필터 |
| **정렬** | 이름 등 컬럼 기준 정렬 |
| **새 사용자 추가** | 이름, ID, 입사일, 부서, 직급, 직책, 권한 등 입력 후 저장 |
| **사용자 수정** | 행 클릭 시 수정 모달에서 부서·직급·직책·권한 등 변경 |
| **휴가 이력** | 각 행의 «휴가이력» 버튼으로 해당 인원 휴가 사용 이력 모달 표시 |
| **삭제** | 사용자 삭제 (권한에 따라 노출) |
| **페이지네이션** | 50명 단위 등 페이지 이동 |

권한(관리자 등급)에 따라 수정 가능 범위가 제한됩니다.

### 3-2. 퇴사자

퇴사 처리된 직원을 조회하고, 필요 시 재직으로 복귀시키는 화면입니다.

| 기능 | 설명 |
|------|------|
| **퇴사자 목록** | No, ID, 이름, 부서, 직급, 퇴사일, **휴가 이력** |
| **검색·부서 필터** | 이름/ID 검색, 부서별 필터 |
| **휴가 이력 보기** | 행별 «휴가 이력 보기»로 해당 인원 휴가 사용 이력 모달 |
| **상세정보** | 행 클릭 시 퇴사자 상세 정보 모달 |
| **재직자로 복귀** | 퇴사자를 다시 재직 상태로 변경 |

---

## 4. 선물보내기

직원에게 **선물(상품/쿠폰 등)**을 검색하고 발송하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **상품 검색** | 상품명 등으로 검색 |
| **상품 목록** | 그리드 형태로 상품 표시, 페이지네이션 |
| **선물 보내기** | 상품 선택 후 대상 사용자 지정해 발송 |
| **발송 이력** | 발송한 선물 내역 조회 (화면 구성에 따라) |

외부 선물 API와 연동해 실제 발송이 이루어지는 구조입니다.

---

## 5. 휴가 총괄 관리

휴가 부여·신청·승인·현황을 관리하는 메뉴 묶음입니다.

### 5-1. 부여요청목록

**특별 사유로 인한 휴가 부여 요청**을 검토·승인·반려하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **요청 목록** | 요청자, 사유, 기간, 상태 등 조회 |
| **승인 / 반려** | 요청 건별 승인 또는 반려 처리 |
| **필터·검색** | 상태, 기간, 요청자 등으로 필터 |

### 5-2. 부여내역

**이미 처리된 휴가 부여 내역**을 조회하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **부여 내역 목록** | 처리 완료된 부여 건별 상세 (대상자, 기간, 사유, 처리일 등) |
| **검색·필터** | 기간, 대상자, 부서 등 조건 조회 |
| **메모** | 부여 건별 메모 조회·관리 (지원 시) |

### 5-3. 전사원 휴가현황

**전체 직원의 휴가 잔여일·사용 현황**을 한눈에 보는 화면입니다.

| 기능 | 설명 |
|------|------|
| **휴가 현황 테이블** | 직원별 잔여 연차, 사용일, 부서 등 |
| **부서·연도 필터** | 부서별, 연도별 조회 |
| **정렬** | 잔여일, 사용일 등 컬럼 정렬 |
| **내보내기** | Excel 등으로 내보내기 (지원 시) |

### 5-4. 임의휴가부여

관리자가 **직원에게 직접 휴가를 부여**하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **대상자 선택** | 사용자 검색·선택 |
| **휴가 종류·기간** | 휴가 유형, 시작일·종료일, 일수 입력 |
| **사유** | 부여 사유 입력 (선택) |
| **부여 실행** | 입력 내용으로 휴가 부여 처리 |

### 5-5. 권한관리자 지정

**휴가 승인 권한을 가진 관리자(승인자)**를 지정·관리하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **승인자 목록** | 현재 지정된 승인자 목록 조회 |
| **추가** | 새 승인자 지정 (사용자 선택) |
| **제거** | 승인자 권한 해제 |

### 5-6. 휴가신청승인 관리

**직원이 신청한 휴가**를 조회하고 승인·반려하는 화면입니다.

| 기능 | 설명 |
|------|------|
| **신청 목록** | 신청자, 휴가 종류, 기간, 상태 등 조회 |
| **승인 / 반려** | 건별 승인 또는 반려 |
| **필터** | 기간, 상태, 부서, 신청자 등 |
| **상세** | 신청 내용 상세 보기 |

휴가 관련 데이터는 외부 휴가 API와 연동됩니다.

---

## 6. 공통 동작

- **로그인/로그아웃**  
  - 로그인 후 JWT 등으로 세션 유지, 로그아웃 시 토큰 삭제 후 로그인 화면으로 이동.
- **권한**  
  - 관리자 등급(0~5 등)에 따라 메뉴 노출·수정 가능 범위가 다르게 적용됩니다.
- **반응형**  
  - 홈·재직자·퇴사자·휴가 화면 등은 브라우저/창 크기에 맞춰 레이아웃이 조정됩니다.
- **내보내기**  
  - 앱 버전 내역, 생일자 동의, 추석 선물, 휴가 현황 등 여러 화면에서 Excel·Markdown 내보내기를 지원합니다.

| 메뉴 | 화면 | 한 줄 설명 |
|------|------|------------|
| **홈** | index | 총 사용자·대화·활성 사용자, 미사용자·앱버전·생일자 동의·추석 선물·최근 활동 |
| **대시보드** | dashboard | 기간별 대화량·사용자 통계, 차트 |
| **사용자 관리** | 재직자 | 재직자 CRUD, 휴가 이력 보기 |
| **사용자 관리** | 퇴사자 | 퇴사자 조회, 휴가 이력, 재직 복귀 |
| **선물보내기** | gifts | 상품 검색·선택 후 직원에게 선물 발송 |
| **휴가 총괄** | 부여요청목록 | 특별 휴가 부여 요청 승인/반려 |
| **휴가 총괄** | 부여내역 | 처리된 휴가 부여 내역 조회 |
| **휴가 총괄** | 전사원 휴가현황 | 전 직원 휴가 잔여·사용 현황 |
| **휴가 총괄** | 임의휴가부여 | 관리자가 직접 휴가 부여 |
| **휴가 총괄** | 권한관리자 지정 | 휴가 승인자 추가/제거 |
| **휴가 총괄** | 휴가신청승인 관리 | 직원 휴가 신청 승인/반려 |

이 문서는 **AAA 관리자 홈**의 주요 기능을 설명하기 위한 개요입니다.  
메뉴별 상세 동작은 실제 화면과 권한에 따라 다를 수 있습니다.
